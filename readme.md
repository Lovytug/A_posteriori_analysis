# Проект "Апостериорный анализ"
Проект по моделированию задачи наведения вертолёта на корабль с использованием дискретного фильтра Калмана

Основная цель проекта — реализовать модель движения двух объектов, синтезировать дискретный фильтр Калмана и провести апостериорный анализ ошибок оценивания

Проект реализован на с++ с использованием визуализации результатов моделирования.

Ниже представлены - Сборка проекта, Постановка задачи и необходимы соотношения, Результаты


## Cборка
Этот проект использует CMake для сборки. Есть возможность скомпилировать его одним из двух способов: с автоматической загрузкой Eigen или с использованием локальной установки Eigen.

### Вариант 1: Автоматическая загрузка
Eigen будет автоматически загружен во время конфигурации:

#--bash--#
mkdir build && cd build
cmake .. && cmake --build .

### Вариант 2: Использование локальной установки Eigen
Если Eigen уже установлен в вашей системе:

#--bash--#
mkdir build && cd build
cmake -DEIGEN_USE_LOCAL=ON -DEIGEN_ROOT_DIR=/путь/к/eigen ..
cmake --build .


## Постановка задачи
Рассматривается задача сброса груза с вертолёта на палубу корабля. Положение объектов рассматривается в проекции на плоскость местного горизонта

Особенности задачи:
  - движение вертолёта и корабля подвержено случайному дрейфу
  - дрейф вертолёта обусловлен ветром
  - дрейф корабля обусловлен течением
  - измерения содержат случайные ошибки

На основе измерений требуется:
  - оценить ошибки прицеливания
  - восстановить вектор состояния системы
  - определить статистические характеристики ошибок оценивания


## Математическая модель
В работе используется линейные модели движения объектов. А значит и используется классический линейный фильтр Калмана.

### Модель движения коробля
Положение корабля определяется уравнением:
```
X_B(t{i+1}) = X_B(ti) + delta_t · V_S(ti)
```
где  X_B - вектор положения коробля
     V_S - вектор скорости дрейфа коробля (течения)

### Модель движения вертолета
Положение вертолёта определяется уравнением:
```
X_H(t_{i+1}) = X_H(t_i) + delta_t · (V_H(t_i) + V_w(t_i))
```
где  V_H - вектор управляемой скорости вертолета
     V_w - вектор скорости ветра
Скорость наведения определяется законом:
```
V_H = (X_B − X_H) / (2*delta_t) − V_w + V_S
```
При этом скорость ограничена условием:
```
|V_H| ≤ 250 км/ч
```


## Модель измерений
Измеряется вектор ошибок прицеливания:
```
Y = delta_X + η
```
где  delta_X = X_B - X_H
     η - случайная ошибка измерений
Шум измерений считается гауссовским:
```
η ~ N(0, Kη)
```


## Алгоритм оценивания
Для оценивания состояния системы используется линейный дискретный фильтр Калмана
Алгоритм состоиз из двух этапов

### Прогноз
```
X^крышка(i+1) = Φ X*(i) + V U(i)
P^крышка(i+1) = Φ P*(i) Φ^T + Ψ Dξ Ψ^T
```
### Коррекция
```
X*(i) = X^крышка(i) + P*(i) H^T Dη^(-1) (Y − H X^крышка(i))
P*(i) = (P^крышка^(-1) + H^T Dη(-1) H)^(-1)
```


## Схема моделирования
Алгоритм моделирования включает следующие этапы:
1. задание начальных положений объектов
2. генерация измерений с гауссовским шумом
3. коррекция оценок состояния с помощью фильтра Калмана
4. прогноз следующего состояния системы
5. вычисление управляющей скорости вертолёта
6. обновление положения объектов
7. повторение процесса до достижения цели.

Моделирование завершается, когда ошибка прицеливания становится меньше 1 метра.
