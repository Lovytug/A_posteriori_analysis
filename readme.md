Проект "Апостериорный анализ"
Этот проект использует CMake для сборки. Вы можете скомпилировать его одним из двух способов: с автоматической загрузкой Eigen или с использованием локальной установки Eigen.

Инструкции по сборке
Вариант 1: Автоматическая загрузка (рекомендуется для большинства пользователей)
Eigen будет автоматически загружен во время конфигурации:

#--bash--#
mkdir build && cd build
cmake .. && cmake --build .


Вариант 2: Использование локальной установки Eigen
Если Eigen уже установлен в вашей системе:

#--bash--#
mkdir build && cd build
cmake -DEIGEN_USE_LOCAL=ON -DEIGEN_ROOT_DIR=/путь/к/eigen ..
cmake --build .
